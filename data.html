Data Schema

Each resource entry in our database includes the following fields:

Field Type Description
id number Unique identifier
name string Organization/program name
category string housing | food | health | legal | crisis
county string Wayne | Monroe | Statewide
city string City name
address string Full street address
phone string Phone number
website string URL or N/A
hours string Operating hours
eligibility string Who can access this resource
intake_process string How to access the service
tags string[] Array of keyword tags
last_verified_date string (ISO 8601) Date this entry was last verified
